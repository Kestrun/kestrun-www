A modern hybrid web framework for PowerShell & .NET

What is Kestrun?

Kestrun is a hybrid web framework built on ASP.NET Core (Kestrel), extending it with PowerShell, C#, and multi-language scripting support. It brings the flexibility of scripting and the performance of .NET into one ecosystem.

Features

  • ⚡ Multi-language routes (PowerShell, C#, F#, VB.NET, JS, Python, …)
  • 🔒 Built-in authentication & certificate management
  • 📄 Razor pages backed by PowerShell scripts
  • 📊 Logging, scheduling, caching, compression
  • 🛠 Designed for extensibility & cross-platform use
